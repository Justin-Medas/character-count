    buttons

    Generates base rules for button styles. $size indicates padding and font size $type defines the background & text colors for all states

    mixins

    button

    @mixin button($size: null, $type: null, $init-defaults: false) { ... }

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Size of the button often pertaining to the outer paddings, and button font size

    		Stringnull
    $type

    A type of button indicated by the style guide. These will change between projects, and more could be added. Most often it describes the button's color, background-color, border-color, and hover/focus state rules

    		Stringnull
    $init-defaults

    Add all the baseline styles share by all button types and sizes.

    		Stringfalse

    Example

    By default the options are large, small, tiny

    .btn {
  @include button($size: large);
}

    By default there are default, alternate, button, and form-inputs options.

    .btn--alternate {
  @include button($type: alternate);
}

    This can contain styles like transition, font-family, text-transform, etc.

    button {
  @include button($init-defaults: true);
}

    mixins

    card

    @mixin card($size: null, $custom-size: null, $orientation: 'stacked', $xy-column-value: 6, $align: center, $justify: center, $collapse: false, $offset-value: -$spacing--2xlarge) { ... }

    Description

    Implement a card system that leverages the style guide

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    large, medium, small -- Refers to spacing / padding around the inner card content. In some cases (often based on deisgn scheme) these sizes also refer to spacing & font sizes

    		Stringnull
    $custom-size

    Custom padding values applied to .card__content__inner

    		Numbernull
    $orientation

    'inline', 'offset', or 'stacked' - Whether the .card__media and card__content elements should be in a column or a row

    		String'stacked'
    $xy-column-value

    The size of your cell. Can be full (100% width), auto (use all available space), shrink (use only the required space) or any fraction (6, 50%, 1 of 2 or 1/2...)

    		Number6
    $align

    passed to the rule align-items: value; flex align the card media and content

    		Stringcenter
    $justify

    passed to the rule justify-content: value; flex justify the card media and content

    		Stringcenter
    $collapse

    true/false Determines if the card content should have left/right/bottom padding

    		Booleanfalse
    $offset-value

    If $orientation is set to offset this value is applied to .card__content to stagger it and the card__media

    		Number-$spacing--2xlarge

    Example

    Set large, medium, or small to .card or any card modifier

    .card {
  @include card($size: 'large');
}

    Set top, right, bottom, left padding to .card or any card modifier

    .card {
  @include card($custom-size: rem-calc(40 30 20 30));
}

    Set 'inline', or 'stacked' to .card or any card modifier

    .card {
  @include card($orientation: 'stacked');
}

    Set large, medium, small .card or any card modifier

    .card {
  @include card($xy-column-value: 50%);
}

    Set any flex alignment values to .card or any card modifier

    .card {
  @include card($align: center);
}

    Set any flex alignment values to .card or any card modifier

    .card {
  @include card($justify: flex-end);
}

    Set true/false to .card or any card modifier

    .card {
  @include card($justify: flex-end);
}

    Set true/false to .card or any card modifier

    .card {
  @include card($orientation: 'offset', $offset-value: -60px);
}
